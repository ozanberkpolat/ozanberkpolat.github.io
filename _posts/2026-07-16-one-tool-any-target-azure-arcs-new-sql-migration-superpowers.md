**Cloud migration just got a whole lot simpler.** For data engineers and IT architects, the challenge of moving SQL workloads often feels like a high-stakes choice between managed simplicity and infrastructure flexibility. Today, Azure Arc bridges that gap by unifying the migration experience across your entire hybrid estate, ensuring your path to the cloud is as smooth as possible.

The Azure Arc migration solution has officially expanded to support **SQL Server on Azure Virtual Machines** as a migration target. This means you now have a single, consistent workflow to move your Arc-enabled SQL Server instances to either **Azure SQL Managed Instance** or **SQL Server on Azure VMs**. By removing the need for disparate tools and fragmented processes, this update allows you to modernize your data estate on your own terms with total confidence.
