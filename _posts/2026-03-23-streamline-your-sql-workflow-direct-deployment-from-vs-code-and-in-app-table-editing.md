Tired of clunky database deployment processes? Microsoft just supercharged Azure SQL with two game-changing features that will revolutionize your workflow. Now you can publish SQL database projects directly from Visual Studio Code, eliminating the need for complex deployment scripts and reducing your deployment time dramatically.

But that's not all - the new in-app table data editor lets you view and modify SQL table data directly within your development environment. No more switching between tools or writing tedious SELECT queries just to make quick data changes. These updates bring database management closer to your fingertips than ever before.
